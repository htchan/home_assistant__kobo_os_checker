# Home Assistant - Kobo OS Check integration
This integration will fetch data from `api.kobobooks.com` to check the latest OS release version of specific Kobo device

## Home Assistant Entity
- kobo_{device_name}__os_version
- Kobo_{device_name}__os_release_date
- kobo_{device_name}__os_release_note_url
